# VPN-Privacy-and-Security-Setup
VPN Privacy and Security Setup is an Hands-on implementation of a Virtual Private Network (VPN) using ProtonVPN to understand encryption, IP masking, and online privacy protection. Includes setup steps, screenshots, detailed report (PDF &amp; DOCX), and VPN configuration summary for internship documentation.

# VPN Privacy and Security Setup

### Internship Task 8 - Cybersecurity Internship (Elevate Labs)

## Objective
To understand how VPNs protect online privacy and enable secure communication over public networks.

## Tools Used
- **VPN Client:** ProtonVPN (Free Tier)
- **Browser:** Google Chrome
- **Verification Tool:** [whatismyipaddress.com](https://whatismyipaddress.com)

## Steps Performed
1. **Selected VPN Provider:** Chose ProtonVPN for its secure free-tier service.  
2. **Downloaded & Installed:** Installed ProtonVPN desktop client from the official website.  
3. **Connected to Server:** Connected to Japan server using ProtonVPN.  
4. **Verified IP Change:** Checked new IP on [whatismyipaddress.com](https://whatismyipaddress.com).  
5. **Tested Secure Browsing:** Confirmed encrypted HTTPS browsing.  
6. **Disconnected & Compared:** Observed IP revert and speed difference.

## VPN Features
- AES-256-bit Encryption  
- OpenVPN & IKEv2/IPSec protocols  
- Kill Switch & DNS Leak Protection  
- No-Logs Policy  

## Benefits
- Protects online identity and hides IP.  
- Encrypts traffic on public Wi-Fi.  
- Prevents ISP and tracker monitoring.  
- Enables access to restricted content.

## Limitations
- Reduced browsing speed.  
- Limited server options on free plans.  
- Some websites block VPN IPs.
